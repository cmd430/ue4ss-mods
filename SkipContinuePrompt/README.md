# Skip Continue Game Confirmation
Skip Continue Game Prompt

## Features
- Continue Game without Confirmation Prompt

## Install
 Put the mod into the `OblivionRemastered\Binaries\Win64\ue4ss\Mods` directory

```
OblivionRemastered
└───Binaries
    └───Win64
        └───ue4ss
            └───Mods
                └───SkipContinue
                    │   Readme.md
                    │
                    └───Scripts
                        │   config.lua
                        │   lib.lua
                        │   main.lua
```
